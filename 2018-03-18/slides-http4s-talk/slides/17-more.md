## Still much more

[Http4s](http://http4s.org/) comes with many other features that don't fit in a 45 mins talk:

- HttpClient[F]
  - Streaming (bracketed)
- HttpMiddleware[F]
  + CSRF
  + GZip
  + Metrics (v0.19)
- Authentication
  + Basic Auth
  + Digest Auth
  + Tsec Librabry (3rd Party)
- SSL / TLS
- Http/2 Support
